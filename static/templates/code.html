It's Open Source


It's fun to start from scratch and build something new using new tools. It's a great way to learn. OpenVote was developed by Dan and Doug this way. There are a lot of new frameworks we wanted to try. We started with Django for server API and Backbone.js for the client, but then saw angular.js on hackernews and refactored the client side of the application using that. It's also the deepest integration with Google Maps API v3 that we've done, and has some intelligence behind the scenes inspired by the local AI Meetup.

Help us make OpenVote even better by contributing to the source code.
